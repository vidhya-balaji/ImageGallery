Image Gallery
This is a simple, read-only image gallery that allows users to view images in an organized format. The gallery provides an intuitive interface to explore images without the ability to upload, edit, or delete them. This is ideal for showcasing images in a read-only environment, such as a portfolio, a product showcase, or a gallery display.

Features
View Only Mode: Users can browse and view images, but cannot modify, upload, or delete them.
Responsive Design: The gallery adapts to different screen sizes, providing a seamless experience across devices.
Image Thumbnails: Displays image thumbnails in a grid format, making it easy to find and view images.
Lightbox View: Clicking on an image opens it in a larger view for a better visual experience.
Navigation Controls: Users can easily navigate through images with a simple, intuitive interface.
